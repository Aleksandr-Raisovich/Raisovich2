Как это работает

UI/UX Pro Max — это скилл для ИИ-ассистентов, который автоматически подбирает оптимальные дизайн-решения на основе ваших требований.

  • Анализирует требования проекта
  • Подбирает стиль, палитру и шрифты
  • Применяет правила рассуждения для оптимального выбора
  • Предоставляет готовый CSS и рекомендации
Терминал 
# Генерация дизайн-системы
python3 skills/ui-ux-pro-max/scripts/search.py \
  "beauty spa wellness" --design-system \
  -p "Serenity Spa"

# Поиск по домену
python3 skills/ui-ux-pro-max/scripts/search.py \
  "glassmorphism dark" --domain style

# Рекомендации по стеку
python3 skills/ui-ux-pro-max/scripts/search.py \
  "layout responsive" --stack html-tailwind
